How to use Agentic RAG

×
Getting started
📁
Upload your documentsGo to "Upload Docs" and drag in a .txt, .md, or .pdf file. It gets indexed automatically for search.
💬
Ask anythingType a question in the chat. The system decides automatically: search your docs, browse the web, or use conversation memory.
✅
Verified answers onlyA Critic agent validates every answer before delivery. Unverified answers are retried up to 3 times automatically.
Smart helper buttons
📋
SummarizePrepends a summarization instruction — perfect for quick overviews of long documents.
•
Bullet pointsReformats the expected answer as a clean bullet list for easy scanning.
⚖
CompareFrames the question around contrasting multiple topics, documents, or approaches.
⚠
Find risksFocuses the answer on limitations, drawbacks, and potential issues.
→
Next stepsExtracts recommended actions or follow-on tasks from the content.
How routing works
🗂
RAG routeQuestions about your uploaded files — uses Hybrid Search (Dense + BM25) with Cross-Encoder reranking.
🌐
Web routeQuestions requiring current data — answered via Tavily real-time web search.
🧠
Memory routeReferences to earlier conversation — answered instantly from session memory.
